RMS Servers

The record management system (RMS) performs a wide range of functions for the King County Police Department such as processing case documents, housing criminal records, registering sex offenders and more. In order to run the RMS and the computers that interact with the system, servers must be active.

Since there are many ways to attack the servers, they present one of the biggest security risks for the King County Police Department. Keeping the servers active is also essential because the RMS serves such a large portion of the police department's functions.


Threats

Unused Services and Open Ports

When a server administrator sets up a web server, there are a number of default applications that are run on the server. Some of these services can cause extra web traffic on your server and cause it to bottleneck. Other applications may leave a hole for attackers to get in and cause havok.

DOS Attacks

A denial of service (DOS) attack occurs when a server is intentionally overloaded with web requests to slow down traffic. Denial of service attacks are generally carried out to prevent the server from servicing customers/users.

In the context of the King County Police Department, servers provide information to employees about criminal records and crime reports, so keeping them active is absolutely vital to the operations of the police department.


Mitigations

Close Unused Ports

One solution to mitigating the risk presented by open ports and unused services is to close the ones that are not needed. Keeping unneeded default services open will only open avenues for attackers to breach the servers. Instead, server admins should try to keep only the vital services and ports open and close all others.

Firewalls

To prevent denial of service attacks originating from a single computer, a web application firewall is a viable option. A web application firewall will prevent a denial of service attack by inspecting HTTP requests and checking the packets against a set of rules to accept or deny the packets.

Having some type of enterprise firewall should help filter out these smaller denial of service attacks.
